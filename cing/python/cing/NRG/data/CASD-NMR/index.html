CASD-NMR CING Validation Reports

These reports are based on the data send in by authors for CASD-NMR. The image itself links to the entry's CING validation report. Please reference the relevant papers.

Additional local resources:
  • A list of all entries can be found here. The entries that end with "Org" are the ones solved by the contributors whereas the entries that end with a city's name are the automated predictions. Some city names are followed by a numeral to distinguish between (groups of) different investigators within the same lab.
  • The results per target can be found here.
  • A spreadsheet of more info is presented here (CSV) and here (Mac Numbers '09 format).
  • A CSV dump files of the Postgresql RDB is present here. The database is being filled with entry, chain, residue, & atom validation data.
  • You may have to dig into the data; e.g. to find the summaries for any one entry go here where R3 are the 2nd & 3rd characters of the entry AR3436ACheshire.
  • The validation CING log file can be found like here sorted by date & time of when the validation was done.
  • The preparation of the data was carried out automagically by scripts using functionality in Wim Vranken's FormatConverter and CING. The log files of this preparation can be found like here again just take the latest version of it.
