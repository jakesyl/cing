NRG-CING Validation Reports

These reports are based on the coordinate data in wwPDB combined with 'converted' data in the NMR Restraints Grid (NRG) where possible. The image itself links to the entry's CING validation report. The PDB entry link is to the RCSB PDB site; the ePDB site is here. Below that, a link to the BMRB entry is given for those PDB entries originating from NMR studies that have NMR experimental data deposited with BMRB. Please reference the relevant papers.
