NRG-CING is an resource of CING validation reports are based on the coordinates in wwPDB combined with 'converted' data in the NMR Restraints Grid and chemical shifts from BMRB for biomolecular structures created by NMR spectroscopy. The PDB identifier links to the specific CING validation report.
