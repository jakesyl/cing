Glossary

Below is a growing list of terms used inside the CING reports. Please ask us about missing information and we will add it to this glossary.

Key Explanation
csdchemical shift difference
cvcircular variance
d1d2This parameter describes the CA-CB vector of residue i-1 with respect to the CA-CB vector of residue i. The plot of residue i gives two of those combined; i.e. (CA-CB)(i-1)-(CA-CB)(i) versus (CA-CB)(i)-(CA-CB)(i+1). Hence, it should be read as the conformation of residue i relative to its two neighbours i-1 and i+1. We still need to properly asses its usefulness for structure validation purposes, but limited manual assessment indicates it to be useful in identifying potential problem areas.
distance class: intra-residualThose distance restraints (DR) that are within the same residue
distance class: sequentialDRs between direct neighbouring residues i and i+1
distance class: medium-rangeDRs between residues i and i+n with n in the range [2,4]
distance class: long-rangeDRs between residues i and i+n with n over 4. This class includes the inter-chain DRs if present.
distance class: ambiguousDRs with more than one possible pair excluding consideration of ambiguity over stereospecific assignment (SSA).
drdistance restraints
rmsdroot mean square deviation
rogred orange green. The ROG score is used to aid in identifying bad, poor, and normal entities such as residues, chains, molecules.
ssastereospecific assignment
